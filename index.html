Comandos basicos de GIT


  • git init: para decir "aqui voy a comenzar a usar GIT", aqui estaremos apenas en el workin directory
  • git status: para ver el estado de nuestros archivos
  • git add <archivo>: para agregar una archivo al staging area.
  • git log: para ver un historias de los commits que se han hecho
  • git checkout <archivo>: para regresar al ultimo commit realizado
  • git diff <archivo>: para ver el codigo modificado
  • git commit -m "description": para hacer un commit de una manera mas rapida
  • git branch: para ver los branches disponible
  • git branch -r: para ver los branches disponibles del repositorio remoto
  • git branch -a: para ver todas las ramas disponibles tanto locales como remotas
  • git branch <nombre-branch>: para crear un nuevo branch
  • git checkout -b <nombre-branch>: para crear un nuevo branch
  • git branch -M <nombre-branch>: para cambiar el nombre del branch
  • git checkout <nombre-branch>: para cambiar de branch
  • git add . :Agrega todos los archivos al staging area
  • git remote add origin <url-git>:para indicar a que repositorio se estaran subiendo los cambios
  • git remote -v: para ver que hay configuradoen git remote
  • git push -u origin <branch-name (where you want to push from - local branch name) >: sube los cambios que estan de manera local a un repositorio remoto
  • git pull origin: trae los cambios mas recientes que se han hecho en github
  • git merge <nombre-del-branch-a-unir>: para hacer un merge debemos ir al branch y ejecutar este comando
  • git fetch origin: Para traer los cambios mas recientes del repo "origin".
  • git log --graph --pretty=oneline --abbrev-commit : Para ver el grafico en la terminal
  • git push <nombre-remoto> <nombre-rama-local>:<nombre-rama-remota> Para ver el grafico en la terminal
  • git branch -d <nombre-rama> para eliminar una rama (primero hay irse a otra rama)
  • git push origin --delete <mi-rama> para eliminar remota
  • git fetch --prune para actualizar las ramas que fueron eliminadas pero aun quedan en cache local
